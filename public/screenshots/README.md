# Screenshot Evidence Library

This directory stores real product screenshots for AI Builder Hub evidence updates.

Rules:

- Do not add fake screenshots.
- Do not use stock images.
- Do not expose private account data, billing details, API keys, email addresses, account IDs, or customer data.
- Record verification dates in `src/data/posts.ts` when a screenshot is added.
- Keep filenames simple, such as `dashboard.png`, `pricing.png`, `billing.png`, `deployment.png`, `setup.png`, and `analytics.png`.

Current providers:

- `hostinger/`
- `namecheap/`
- `vultr/`
- `semrush/`
