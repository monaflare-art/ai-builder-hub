## Tencent Cloud CPS

### Verified Registration Path

- Open [Tencent Cloud CPS](https://cloud.tencent.com/act/partner/cps).
- Use a Tencent Cloud account that has completed personal real-name verification.
- Click "我要推广".
- Fill the promoter application information.
- After application success, enter the promotion backend and copy approved promotion links/materials.

### Verified Requirements

- Personal real-name Tencent Cloud account is required.
- Enterprise users cannot become promoters.
- Agents, Tencent employees, and partner employees are excluded by the public rules.
- Bank account information is required for commission settlement.
- Promotion must not use illegal, misleading, hijacking, forced-bundling, or self-referral behavior.

### Payment and Settlement

- Commission is not withdrawn manually from the backend.
- Tencent Cloud calculates the previous month commission during the next month.
- Public rules state payment is made by Tencent Cloud financial system to the promoter's backend bank account.
- Tencent withholds applicable taxes.

### Owner Execution Notes

OWNER MUST FILL:

- Tencent Cloud login.
- Personal real-name verification.
- ID verification.
- Phone verification.
- Bank account.
- Any tax or payout information.

## Alibaba Cloud Ambassador

### Verified Registration Path

- Open [Alibaba Cloud Cloud Ambassador](https://dashi.aliyun.com/).
- Log in with an Alibaba Cloud account.
- Join through the Cloud Ambassador page or other official Cloud Ambassador entry.
- Accept the Cloud Ambassador rules and integrity terms.

### Verified Requirements

- Alibaba Cloud personal or enterprise real-name verification is required.
- Public agreement states the account must agree to the Cloud Ambassador rules.
- Alibaba Cloud can determine that an account is unsuitable for joining.
- Alibaba employees, outsourcing staff, and certain distribution/partner sales identities are excluded.

### Payment and Settlement

- Rewards are measured in 云气.
- Public FAQ states cloud ambassador reward value is calculated at 1 RMB paid amount = 100 云气, and 100 云气 = 1 RMB for cash withdrawal.
- One valid order is enough to withdraw.
- Personal users are responsible for applicable taxes; Alibaba Cloud withholds tax during withdrawal.
- Enterprise users follow the enterprise withdrawal process and bind enterprise payout information.

### Owner Execution Notes

OWNER MUST FILL:

- Alibaba Cloud login.
- Personal or enterprise real-name verification.
- ID verification.
- Phone verification.
- Withdrawal account.
- Tax or enterprise payout details if requested.

## Baidu AI Cloud Ambassador

### Verified Registration Path

- Open [Baidu AI Cloud Ambassador](https://cloud.baidu.com/campaign/ambassador/index.html).
- Use the official application form: [Baidu AI Cloud Ambassador application form](https://cloud.baidu.com/campaign/ambassador-recruitForm/index.html).
- Submit the required recruitment/application information.

### Verified Requirements

- Public recruitment materials describe individual applicants as eligible.
- Public payout rules require commercial information verification before bank payout.
- Correct bank card information must be bound for reward payout.
- Same-person/self-referral, refund abuse, and agent-related conflicts can invalidate rewards or cancel eligibility.

### Payment and Settlement

- Eligible new-customer orders are counted within the 30-day recommendation cycle.
- Public rules state monthly rewards are counted on the 26th of the following month.
- Payout is generally made after the platform completes settlement.
- Public rules state rewards are sent to the bank card that has passed commercial information verification.
- Reward payout is split into four months: 20%, 20%, 20%, and 40%.
- Baidu AI Cloud withholds applicable taxes.

### Owner Execution Notes

OWNER MUST FILL:

- Baidu account login.
- Name/contact fields if requested.
- ID verification.
- Phone verification.
- Commercial information verification.
- Bank card information.
- Any tax or payout information.
