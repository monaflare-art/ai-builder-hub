## Evidence Standards Used

- Builder Experience Notes
- Performance Notes
- Pricing Change Notes
- Deployment Notes
- Feature Evidence
- Pros Evidence
- Affiliate/Billing Evidence

Required screenshot sections:

- Dashboard Screenshot
- Pricing Screenshot
- Deployment Screenshot
- Billing Screenshot
- Alternatives Screenshot or comparison-table visual

Priority score:

- 5 = direct money page with immediate affiliate conversion impact and high evidence need
- 4 = strong commercial page with important credibility gaps
- 3 = useful page, but evidence can follow after higher-intent reviews

## Cross-Page Findings

- All five rewritten pages contain the baseline evidence scaffold.
- All five pages still need real screenshots before claims should become stronger.
- No page currently has `Last reviewed`, `Last verified`, `Pricing checked`, or `Screenshot updated` metadata inside the content model.
- Evidence is currently embedded as article body text, which is fast but not scalable for tracking verification freshness.
- Hosting, domain, VPS, and SEO pages need different screenshot checklists; one generic evidence block is not enough for long-term reviews.
