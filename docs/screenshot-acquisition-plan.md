# Screenshot Acquisition Plan

Scope: first evidence pass for the five rewritten money pages.

No screenshots are added in this phase. This document defines what to capture later, why it matters, and capture priority.

Priority:

- P0 = needed before stronger affiliate CTA or recommendation language
- P1 = important credibility support
- P2 = useful but can wait

## Screenshot Handling Rules

- Do not expose personal data, billing details, account IDs, API keys, email addresses, payment methods, or customer data.
- Crop or blur sensitive fields before publishing.
- Capture current pricing only with the capture date recorded.
- Prefer official dashboard and pricing pages over marketing images.
- Add captions that explain what the screenshot proves.
- Refresh screenshots when pricing, dashboard UI, or setup flow changes.
