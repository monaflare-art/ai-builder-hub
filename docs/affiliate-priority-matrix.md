# Affiliate Priority Matrix

Last reviewed: 2026-06-18

Priority is based on:

- Program availability
- Commercial intent fit
- Existing review/comparison coverage
- Likelihood of approval for a new but focused content site
- Commission potential
- Evidence/readiness gaps
